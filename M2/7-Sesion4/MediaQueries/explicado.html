Media Queries

Las Media Queries son una técnica en CSS que permiten aplicar estilos específicos a elementos de una página web dependiendo de ciertas características del dispositivo, como el ancho de la pantalla, la resolución, la orientación, y más. Esto permite que un diseño se adapte de manera óptima a diferentes dispositivos, mejorando la experiencia del usuario.

¿Cómo funcionan?

Las Media Queries funcionan evaluando condiciones específicas y aplicando estilos CSS si esas condiciones se cumplen. Por ejemplo, puedes cambiar el layout de una página dependiendo del ancho de la pantalla.

Ejemplo: Ajustar el fondo y el color del texto según el ancho de la pantalla.

        /* Estilos por defecto */
        body {
        background-color: white;
        color: black;
        }

        /* Media Query para pantallas de 600px o más */
        @media (min-width: 600px) {
        body {
            background-color: lightblue;
            color: white;
        }
        }

Tipos de Media Queries

Algunas de las Media Queries más comunes incluyen:

  • min-width / max-width: Aplica estilos según el ancho mínimo o máximo del viewport.
  • min-height / max-height: Aplica estilos según la altura mínima o máxima del viewport.
  • orientation: Aplica estilos según la orientación del dispositivo (landscape o portrait).
  • prefers-color-scheme: Aplica estilos según la preferencia de esquema de color del usuario (light o dark mode).

Las Media Queries son fundamentales para crear diseños responsivos que se adapten a cualquier dispositivo y resolución.
